Protótipo: sugestões
Tela onde o técnico pode o sistema mostra sugestões de duplas para cada atleta que o técnico cadastrou.
No protótipo mostra só um atleta cadastrado (João Henrique), mas poderia mostrar vários.
Clicando nos nomes com underline seria aberta uma janela (modal) detalhando o atleta / técnico.
Não sei como a informação da competição entra na tela. Talvez a tela sempre mostra sugestões de uma só competição (então é aberta por algum link tipo "ver sugestões da competição X"). Talvez os atletas são agrupados por competição. Talvez cada competição fica numa aba diferente. Talvez cada atleta tem um campo competição (poderia ficar repetitivo).
Se a cada competição um técnico cadastra poucos atletas, talvez nem vale a pena ter uma tela para o sistema sugerir, o próprio técnico pode buscar outros atletas por meio de uma tela de consulta.
Mesmo assim acho que dá pra reutilizar algumas ideias desse protótipo para outros.
